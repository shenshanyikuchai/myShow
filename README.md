# 河南创亿彩灯艺术有限公司 — 企业官网

彩灯艺术 · 军事模型 · 文旅亮化 — 科技风响应式静态站点。

## 本地预览

```bash
python3 -m http.server 8765
# 浏览器打开 http://localhost:8765
```

## 部署

本仓库可通过 [GitHub Pages](https://pages.github.com/) 托管：Settings → Pages → Source 选择 `main` 分支根目录。

## 图片资源

- 网站运行时仅使用 `assets/images/` 本地图片，不依赖外网 CDN。
- 下载真实产品图（在系统终端执行）：

```bash
python3 scripts/download-real-images.py
```
